# browser-use_poc
browser-use poc

Create a virtual environment :
```bash
python -m venv .venv
.\.venv\Scripts\activate
```

Install the required dependencies in the virtual environment :
```bash
pip install -r requirements.txt
```

Install playwright dependencies in the virtual environment :
```bash
playwright install
```
